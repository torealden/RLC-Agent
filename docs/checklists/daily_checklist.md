# Daily Operations Checklist

## Morning Brief (Before Market Open — 7:00 AM ET)

### System Check
- [ ] Review DOC summary from prior evening
  - Data collection status (all green?)
  - Anomalies flagged (price moves, data gaps)
  - Crush margins (any significant change?)
  - Feedstock allocation (any shifts in optimal mix?)
- [ ] Check dispatcher heartbeat — is the system running?
- [ ] Review overnight news for market-moving events
  - Geopolitical (Iran, Black Sea, trade policy)
  - Weather (US crop belt, Brazil, Argentina)
  - Policy (EPA, LCFS, 45Z developments)

### Data Verification
- [ ] Futures prices updated (ZS, ZL, ZM, ZC, ZW, CL, HO, RB)
- [ ] Cash prices updated (USDA AMS)
- [ ] Any weekly report due today? (see event calendar below)

### Analytical Priorities
- [ ] What report is the market focused on this week?
- [ ] Any balance sheet adjustments needed based on overnight developments?
- [ ] Client communication needed? (alert, update, scheduled call)
